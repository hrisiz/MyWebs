How to Add / Edit News
1. go to config folder
2. open news.php
3. edit the array as shown

1. отидете config папката
2. отворете news.php
3. редактирайте на масива, както е показано


config/news.php 
	$option['news'] = array(
	
		copy this:
		
		array(
			'name' => 'some news',
			'description' => '
				some text
			'
		),
		
		paste and edit:
		
		array(
			'name' => 'enter your title news here',
			'description' => '
				enter your news here
			'
		),
		
		
	);
